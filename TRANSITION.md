# Transition Guide

Stubborn Contract is the official continuation of Spring Cloud Contract, led by
[Marcin Grzejszczak](https://toomuchcoding.com), one of its original creators.

**Current status:** the project has moved under the Stubborn organization. The first
Stubborn Contract release, including new Maven coordinates, package names, and migration
instructions, is planned. Until then, existing users can continue using their current
Spring Cloud Contract versions while preparing for the transition.

- **What happened?** Spring Cloud Contract is entering a new chapter as it transitions
  away from the Spring Cloud portfolio. This repository is its new home.
- **What changes?** A move toward Stubborn-branded Maven coordinates and a package
  rename from `org.springframework.cloud` to `sh.stubborn`, planned but not yet complete.
- **What stays the same?** The contract-testing approach and the Apache 2.0 license.
- **What should I do now?** Keep using your current Spring Cloud Contract version.
  Nothing breaks today.
- **Where will migration instructions appear?** Here, and on the website, once the
  first Stubborn Contract release ships.

Full FAQ: https://stubborn.sh/transition-guide
